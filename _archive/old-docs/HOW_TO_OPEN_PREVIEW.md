# How to Open the Sandbox Preview in Your Browser

## Method 1: Double-Click (Easiest) ⭐

1. **Open File Explorer** (Windows key + E, or click the folder icon on your taskbar)
2. **Navigate to:** `C:\program-loader-seed\`
3. **Find the file:** `sandbox-preview.html`
4. **Double-click** the file
5. It should open automatically in your default browser!

## Method 2: Right-Click Menu

1. **Open File Explorer**
2. **Navigate to:** `C:\program-loader-seed\`
3. **Right-click** on `sandbox-preview.html`
4. **Select:** "Open with" → Choose your browser (Chrome, Edge, Firefox, etc.)

## Method 3: Drag and Drop

1. **Open your browser** (Chrome, Edge, Firefox, etc.)
2. **Open File Explorer**
3. **Navigate to:** `C:\program-loader-seed\`
4. **Drag** `sandbox-preview.html` **into** the browser window
5. The preview will load!

## Method 4: From Browser Address Bar

1. **Open your browser**
2. **Click the address bar** (where you type URLs)
3. **Type:** `file:///C:/program-loader-seed/sandbox-preview.html`
4. **Press Enter**

## Method 5: From Command Line

1. **Open PowerShell or Command Prompt**
2. **Type:**
   ```
   start sandbox-preview.html
   ```
3. **Press Enter**

## Quick Tips

- **Default browser:** If you double-click, it opens in your default browser
- **Choose browser:** Right-click → "Open with" → pick a specific browser
- **File location:** The file is at `C:\program-loader-seed\sandbox-preview.html`
- **No internet needed:** This file works offline - no internet connection required!

## What You'll See

Once opened, you'll see:
- ✅ A beautiful preview of the tool rotation system
- ✅ "Tool of the Day" section showing today's rotating tool
- ✅ Upcoming schedule showing which tools will be featured
- ✅ Test controls to try different dates

## Need Help?

If the file doesn't open:
- Make sure you're looking in the right folder: `C:\program-loader-seed\`
- Try right-clicking and choosing "Open with" → your browser
- Make sure you have a browser installed (Chrome, Edge, Firefox, etc.)
